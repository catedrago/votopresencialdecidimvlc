Archivo de configuración

    • Configuración del USB
    Se va a descargar el siguiente archivo:
    • 1. Archivo configuración.json con los datos necesarios para iniciar el sistema.
    Una vez descargados es necesario copiarlos en el dispositivo USB preparado.
    Archivos descargados
    Es necesario formatear el USB que se va a utilizar antes de conectarlo al sistema
    • 1. Conectar el dispositivo USB en el ordenador.
    • 2. Boton derecho sobre el dispositivo y selecciona "Formatear".
    • 2. Cambiar el tipo de formato a FAT32.
    • 3. Cambiar el nombre del dispositivo a "evotebox".
    • 4. Click en "Formatear".
    Estructura del archivo CSV
    Se necesitan dos archivos CSV, uno contiene los dni del censo y tendrá la siguiente estructura:
    • dni <- no incluido en el csv

      01234517B

      01234567B

      01234567C

    El segundo archivo CSV contiene las propuestas y tendrá la siguiente estructura:
    • titulo,descripcion,presupuesto

      Test,descripcion test,10000

      Piscina, piscina municipal,8900000

      Parque, parque de bolas,300000
